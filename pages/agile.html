Agile Game

This is a game that i did with 2 of my friends.
We made this game in 8 weeks time and we did it with the agile method.
The game is a 2D (pokemon) topdown RPG game.


Now the story about the game.

the big bad boss of the game had cursed magic and cursed everyone into furniture, object and robots.
the main character is a gun with the ability to "eat" other guns and if you "eat" a different gun you can become that gun.
you can move with the help of a robot that esacape the mind controll that every robot is under.
